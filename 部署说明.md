# Codex 自动化 — 下载页部署说明

## 子域名绑定

将 `www/` 目录下的文件上传到你的服务器，绑定子域名即可。

推荐子域名: `app.myonlyheart.xyz`

## Nginx 配置

```nginx
server {
    listen 80;
    server_name app.myonlyheart.xyz;

    root /var/www/codex-automation;
    index index.html;

    # 大文件下载支持
    client_max_body_size 50m;

    location / {
        try_files $uri $uri/ =404;
    }

    # 启用 gzip 对 exe 没用（已压缩），跳过
}
```

## 上传方式

### 方式 1: SCP

```bash
scp www/Codex自动化.exe user@myonlyheart.xyz:/var/www/codex-automation/
scp www/index.html user@myonlyheart.xyz:/var/www/codex-automation/
```

### 方式 2: FTP/SFTP

将 `www/` 下所有文件上传到服务器网站根目录。

## DNS 配置

在 `myonlyheart.xyz` 的 DNS 管理中添加一条 A 记录或 CNAME:

| 类型  | 名称 | 值              |
|-------|------|-----------------|
| A     | app  | 你的服务器 IP    |

或:

| 类型  | 名称 | 值                      |
|-------|------|-------------------------|
| CNAME | app  | myonlyheart.xyz         |

## 访问

上传并配置完成后，用户访问 `https://app.myonlyheart.xyz` 即可看到下载页面。
